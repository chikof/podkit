# Resterm quickstart

1. Run `resterm` in this folder.
2. Press Ctrl+E to switch environments.
3. Open `requests.http`, place the cursor inside a request, then press Ctrl+Enter.
4. Edit `resterm.env.json` or copy from `resterm.env.example.json`.

Next steps:
- Captures let you store response values for later requests.
- Workflows chain requests with shared context.
- `rts/helpers.rts` shows a RestermScript module example.
- See docs in [docs/resterm.md](https://github.com/unkn0wn-root/resterm/blob/main/docs/resterm.md) for details.
